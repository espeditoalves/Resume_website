Resumo do Projeto

Este projeto tem como objetivo realizar uma análise de dados do Spotify, utilizando ferramentas como Python, PySpark, SQL, DBeaver e Power BI.

O projeto foi desenvolvido com os meus próprios dados do Spotify. Esses dados, no formato JSON, foram manipulados e salvos em um banco de dados PostgreSQL. A partir do banco de dados, foi criado um link com o Power BI para o desenvolvimento de um dashboard para análise e visualização dos dados.

Além disso, foi estabelecida uma conexão com o banco de dados via Pandas e PySpark para o desenvolvimento de um notebook com uma Análise Exploratória dos Dados (EDA), permitindo a visualização e exploração detalhada dos dados.

O projeto também conta com uma API do Spotify para a coleta de informações adicionais, que podem ser utilizadas para novas explorações e insights sobre meus gostos musicais.
